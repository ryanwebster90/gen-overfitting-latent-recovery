# Detecting Overfitting of Deep Generators via Latent Recovery

pytorch implementation of "Detecting Overfitting of Deep Generators via Latent Recovery", CVPR, 2019 [arxiv](https://arxiv.org/abs/1901.03396)

# Dependencies

* pytorch
* scipy
* numpy
